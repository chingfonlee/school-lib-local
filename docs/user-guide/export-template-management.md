# 匯出範本管理

## 用途

每年度或縣市的 Excel 採購書單格式若有異動（例如欄位新增、順序調整、表頭文字更改），可透過「範本管理」頁面上傳新的空白範本，讓系統重新偵測欄位位置，後續匯出的採購書單即會採用新格式。

> **範本與書商書單的差異**
>
> | | 說明 | 由誰提供 |
> |---|---|---|
> | **匯出範本** | 空白採購書單格式（Excel 空表） | 主辦縣市或學校自備 |
> | **書商書單** | 書商提供的推薦書目清單 | 書商 |
>
> 此處上傳的是**空白採購書單格式檔（匯出範本）**，不是書商書單。書商書單請至「匯入」頁面的「書商書單」分頁匯入。

---

## 入口位置

導覽列右側 →「**範本管理**」

---

## 支援的範本類型

| 類型 | 說明 |
|------|------|
| 國小 本土文化採購 | 對應 `local_culture` 專案類型 |
| 國小 一般圖書採購 | 對應 `general_books` 專案類型 |
| 國中 本土文化採購 | 對應 `local_culture_jh` 專案類型 |
| 國中 一般圖書採購 | 對應 `general_books_jh` 專案類型 |

---

## 操作流程

### 步驟一：上傳新範本

1. 在「範本類型」下拉選單選擇對應類型（例如：國小 本土文化採購、國中 一般圖書採購）
2. 點選上傳區域或將 `.xlsx` 範本檔拖曳至上傳區
   - **僅支援 `.xlsx` 格式**，不支援 `.xls` 或 `.csv`
3. 確認檔案名稱顯示於上傳區後，點選「**分析欄位**」

### 步驟二：確認欄位對應

系統會自動偵測工作表、標題列與各欄位位置，並顯示偵測結果。

**請人工確認偵測結果是否正確：**

- ✓ 綠色「偵測成功」：欄位位置已自動對應
- ⚠ 紅色「未偵測到」：請從下拉選單手動指定欄
- 若偵測到的欄位有誤，可點選「修改」重新指定

> 標記為「必填」的欄位必須全部對應，否則無法存入。

### 步驟三：存入系統

確認欄位對應無誤後，點選「**確認存入系統**」。

- 存入後立即套用於後續匯出，不影響已匯出的 Excel 檔案
- 範本檔案會保存於 `purchase-order-template/templates/` 資料夾（本機上傳覆蓋用，不納入 git；內建四份範本位於 `purchase-order-template/` 根目錄並納入 repo）
- 每個類型只保留最新一份範本；重新上傳同類型範本會覆蓋舊設定

## 常見問題

### 上傳後「分析欄位」按鈕無法點擊

請確認已同時完成兩項：選擇「範本類型」並上傳 `.xlsx` 檔案。任一未完成時按鈕不會啟用。

### 某欄位顯示「未偵測到」

系統以欄位標題文字自動比對，若範本欄位名稱與預設關鍵字不同，可能無法自動偵測。請從下拉選單手動指定對應的欄。必填欄位全部指定後即可存入。

### 存入後匯出仍顯示格式異常

請確認：
1. 存入時顯示「範本已更新，立即生效」的成功提示
2. 欄位對應結果中必填欄位均已對應
3. 已有選書資料且選書欄位填寫完整
4. 若問題仍存在，可重新上傳範本並再次確認欄位偵測結果
