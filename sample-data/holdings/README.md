# 合成館藏測試資料 / Synthetic Sample Holdings

本目錄包含一份**合成館藏測試資料**，用於搭配 `sample-data/vendor-lists/` 的範例書商書單，測試系統的館藏匯入與書目比對流程。

## 重要說明

- 這**不是**任何學校的真實館藏資料
- 所有書目均為合成資料（公開書目資訊 + 虛構示範書目），不含個資、學校識別資訊、採購記錄
- 館藏條碼（R001–R005、F001–F010）為示範用，非真實圖書館條碼

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `sample-holdings.xlsx` | 合成館藏 15 筆，Sheet：館藏 |

## 資料結構

欄位：`書名`、`作者`、`ISBN`、`出版社`、`館藏條碼`

| 筆數 | 說明 |
|------|------|
| 前 5 筆（R001–R005） | ISBN 取自 `local-culture-vendor-list-2026.xlsx` 中的公開書目，用於示範「已館藏」比對命中 |
| 後 10 筆（F001–F010） | 虛構 ISBN（`978000000000X`）與虛構書名，用於示範其他館藏書目（不在書商書單中） |

## 使用方式

1. 進入「匯入」→ 選擇類型「館藏」
2. 上傳 `sample-holdings.xlsx`
3. 確認欄位對應（系統會自動辨識 `ISBN`、`書名`、`作者`、`出版社`）
4. 匯入後，可至「比對結果」查看前 5 筆（已館藏）與其他書目的比對狀態
